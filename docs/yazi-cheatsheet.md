# Yazi チートシート

> デフォルトキーマップ準拠 (Yazi v26.x, カスタム設定: show_hidden=true, sort_dir_first=true)
> fish abbr: `y` → `yazi`

## 基本

| キー | 操作 |
|---|---|
| `q` | 終了 |
| `Q` | 終了 (cwd-file を出力しない) |
| `Esc` | ビジュアルモード解除 / 選択クリア / 検索キャンセル |
| `~` or `F1` | ヘルプ表示 |
| `w` | タスクマネージャ表示 |
| `Ctrl+Z` | プロセスをサスペンド |

## 検索 / フィルタ / 検出

| キー | 操作 |
|---|---|
| `s` | ファイル名で検索 (fd) |
| `S` | ファイル内容で検索 (ripgrep) |
| `Ctrl+S` | 検索をキャンセル |
| `f` | フィルタ (入力に一致するファイルだけ表示) |
| `/` | 前方検索 (次のマッチへカーソル移動) |
| `?` | 後方検索 |
| `n` | 次のマッチ |
| `N` | 前のマッチ |

> **検索 vs フィルタ vs 検出の違い**
> - `s` / `S` (検索): fd/rg で再帰的にファイルを探してリストを差し替える
> - `f` (フィルタ): 現在のディレクトリ内で表示を絞り込む (ファイルリストは変わらない)
> - `/` (検出): カーソルを移動するだけ (表示は変わらない)

## 入力モード (リネーム、パス入力等)

> Vim ライクな操作。Normal モードと Insert モードがある。

### モード切替

| キー | 操作 |
|---|---|
| `i` | Insert モード (カーソル位置) |
| `I` | Insert モード (行頭) |
| `a` | Insert モード (カーソルの後) |
| `A` | Insert モード (行末) |
| `v` | Visual モード |
| `Esc` | Normal モードに戻る / キャンセル |
| `Enter` | 確定 |

### 移動 (Normal モード)

| キー | 操作 |
|---|---|
| `h` / `l` | 1文字 左 / 右 |
| `b` / `w` / `e` | 単語単位の移動 |
| `0` / `$` | 行頭 / 行末 |
| `^` / `_` | 最初の非空白文字 |

### 編集 (Normal モード)

| キー | 操作 |
|---|---|
| `d` | 選択範囲をカット |
| `c` | 選択範囲をカット + Insert モード |
| `y` | コピー |
| `p` / `P` | ペースト (後 / 前) |
| `u` | Undo |
| `Ctrl+R` | Redo |
| `x` | 1文字カット |
| `r` | 1文字置換 |

### 編集 (Insert モード)

| キー | 操作 |
|---|---|
| `Backspace` | 前の文字を削除 |
| `Ctrl+U` | 行頭まで削除 |
| `Ctrl+K` | 行末まで削除 |
| `Ctrl+W` | 前の単語を削除 |
