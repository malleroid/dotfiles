# Zellij チートシート

> カスタム設定反映済み (Ctrl+Q 無効、Cmd でモード切替、Alt+1-9 タブ切替、compact レイアウト)

## モード切り替え

| キー | モード | 用途 |
|---|---|---|
| `Cmd+G` | Locked | 全キーをアプリにパススルー (vim 等で衝突回避) |
| `Cmd+P` | Pane | ペイン操作 |
| `Cmd+T` | Tab | タブ操作 |
| `Cmd+N` | Resize | ペインリサイズ |
| `Cmd+Shift+H` | Move | ペイン位置移動 |
| `Cmd+S` | Scroll | スクロール・検索 |
| `Cmd+O` | Session | セッション管理 |
| `Cmd+B` | Tmux | tmux 互換操作 |
| `Esc` / `Enter` | Normal | どのモードからでも戻る |

---

## Normal モードで直接使える (モード切替不要)

### ペイン

| キー | 操作 |
|---|---|
| `Alt+N` | 新規ペイン |
| `Alt+H/J/K/L` | フォーカス移動 (左/下/上/右) |
| `Alt+F` | フローティングペイン表示/非表示 |
| `Alt+=/+` | リサイズ拡大 |
| `Alt+-` | リサイズ縮小 |

### タブ

| キー | 操作 |
|---|---|
| **`Alt+1`〜`Alt+9`** | **タブ番号で直接切り替え** |
| `Alt+H/L` | 画面端でタブ切り替え (左/右) |
| `Alt+I` | タブを左に並べ替え |
| `Alt+O` | タブを右に並べ替え |

### レイアウト

| キー | 操作 |
|---|---|
| `Alt+[` | 前のスワップレイアウト |
| `Alt+]` | 次のスワップレイアウト |

## Pane モード (`Cmd+P`)

| キー | 操作 | 自動で Normal に戻る |
|---|---|---|
| `h/j/k/l` | フォーカス移動 (左/下/上/右) | いいえ (連続操作可) |
| `p` | 次のペインへフォーカス順送り | いいえ |
| `n` | 新規ペイン (自動方向) | はい |
| `d` | 新規ペイン (下) | はい |
| `r` | 新規ペイン (右) | はい |
| `s` | 新規スタックドペイン (同じ場所にタブ的に重ねる) | はい |
| `x` | ペインを閉じる | はい |
| `f` | フルスクリーン切替 | はい |
| `z` | ペイン枠線の表示切替 | はい |
| `w` | フローティング表示/非表示 | はい |
| `e` | 埋め込み ↔ フローティング切替 | はい |
| `i` | ピン留め切替 (フローティング専用: 常に最前面に固定。ログ監視等に便利) | はい |
| `c` | ペイン名変更 (サブモードへ遷移) | — |

---

## Tab モード (`Cmd+T`)

| キー | 操作 |
|---|---|
| `n` | 新規タブ |
| `x` | タブを閉じる |
| `c` | タブ名変更 (Pane モードと統一) |
| `h/l` | 前/次のタブ |
| `1`〜`9` | タブ番号で切り替え |
| `Tab` | 直前のタブに戻る |
| `s` | 入力同期 (全ペインに同じ入力) |
| `b` | ペインを新しいタブに分離 |
| `[` | ペインを左の新タブに分離 |
| `]` | ペインを右の新タブに分離 |

---

## Scroll モード (`Cmd+S`)

| キー | 操作 |
|---|---|
| `j/k` | 1行 下/上 |
| `d/u` | 半ページ 下/上 |
| `Ctrl+F` / `PageDown` / `l` / `→` | 1ページ下 |
| `Ctrl+B` / `PageUp` / `h` / `←` | 1ページ上 |
| `s` | 検索開始 → 文字入力 → `Enter` で Search モードへ |
| `e` | エディタでスクロールバック編集 |
| `Ctrl+C` | 最下部に戻って Normal へ |

### Search モード

操作手順: `Cmd+S` → `s` → 検索文字列を入力 → `Enter` で Search モードに遷移。
オプション (`c`/`w`/`o`) は Search モードに入ってから切替える (トグルするたびに結果がリアルタイム更新)。

| キー | 操作 |
|---|---|
| `n` | 次の検索結果 (下方向) |
| `p` | 前の検索結果 (上方向) |
| `c` | 大文字小文字区別トグル |
| `w` | 折返し検索トグル |
| `o` | 全単語一致トグル (ASCII のみ) |
| `j/k/d/u` 等 | Scroll モードと同じスクロール操作 |

## Session モード (`Cmd+O`)

| キー | 操作 |
|---|---|
| `w` | セッションマネージャー (切り替え・作成) |
| `d` | デタッチ (セッションはバックグラウンド継続) |
| `c` | 設定画面 |
| `p` | プラグインマネージャー |
| `l` | レイアウトマネージャー |
| `s` | Share プラグイン |
| `a` | About 画面 |

---

## Resize モード (`Cmd+N`)

| キー | 操作 |
|---|---|
| `h/j/k/l` | 方向指定で拡大 |
| `H/J/K/L` | 方向指定で縮小 (Shift) |
| `=`/`+` | 均等拡大 |
| `-` | 均等縮小 |

---

## Move モード (`Cmd+Shift+H`)

| キー | 操作 |
|---|---|
| `h/j/k/l` | ペインをその方向に移動 |
| `n` / `Tab` | 次の位置へ移動 |
| `p` | 前の位置へ移動 |

---

## Tmux モード (`Cmd+B`)

> tmux 互換のキーバインド。tmux 経験者向け。

| キー | 操作 |
|---|---|
| `"` | 水平分割 (新規ペイン下) |
| `%` | 垂直分割 (新規ペイン右) |
| `h/j/k/l` / `←↓↑→` | フォーカス移動 |
| `o` | 次のペインへフォーカス順送り |
| `z` | フルスクリーン切替 |
| `x` | ペインを閉じる |
| `c` | 新規タブ |
| `,` | タブ名変更 |
| `n` | 次のタブ |
| `p` | 前のタブ |
| `Space` | 次のスワップレイアウト |
| `[` | Scroll モードへ (tmux の copy-mode 相当) |
| `d` | デタッチ |
| `Ctrl+B` | 配下のアプリに Ctrl+B を送信 |

## CLI コマンド

```fish
zellij list-sessions           # セッション一覧
zellij attach <name>           # 再接続
zellij attach --create <name>  # あれば接続、なければ作成
zellij delete-all-sessions -y  # 全セッション削除
zellij action query-tab-names  # タブ名一覧
```

---

## 注意点

- **Ctrl+Q は無効** — 終了は `Cmd+O` → `d` (デタッチ) か `Cmd+Q` (Ghostty 終了)
- **vim/neovim との衝突** — モード切替を Cmd キーに変更済みのため、Ctrl 系との衝突は解消。必要に応じて `Cmd+G` で Locked モードに入る
- **compact レイアウト** — モードのキーヒントが表示されないので、このチートシートを参照
