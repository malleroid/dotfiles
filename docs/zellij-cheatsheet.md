# Zellij チートシート

> カスタム設定反映済み (Ctrl+Q 無効、Cmd でモード切替、Alt+1-9 タブ切替、compact レイアウト)

## モード切り替え

| キー | モード | 用途 |
|---|---|---|
| `Cmd+G` | Locked | 全キーをアプリにパススルー (vim 等で衝突回避) |
| `Cmd+P` | Pane | ペイン操作 |
| `Cmd+T` | Tab | タブ操作 |
| `Cmd+R` | Resize | ペインリサイズ |
| `Cmd+Shift+H` | Move | ペイン位置移動 |
| `Cmd+S` | — | セッションマネージャー (直接起動) |
| `Cmd+Shift+S` | Scroll | スクロール・検索 |
| `Cmd+O` | Session | セッション管理 |
| `Cmd+B` | Tmux | tmux 互換操作 |
| `Esc` / `Enter` | Normal | どのモードからでも戻る |

---

## Normal モードで直接使える (モード切替不要)

### ペイン

| キー | 操作 |
|---|---|
| `Cmd+N` | 新規ペイン |
| `Cmd+H/J/K/L` | フォーカス移動 (左/下/上/右) |
| `Cmd+W` | フローティングペイン表示/非表示 |
| `Cmd+F` | ペイン内検索 (直接検索入力へ) |
| `Alt+=/+` | リサイズ拡大 |
| `Alt+-` | リサイズ縮小 |
| `Alt+P` | スタックドペインのグループに入れる/外す |
| `Alt+Shift+P` | グループ化するペインを一括選択モード |

### タブ

| キー | 操作 |
|---|---|
| **`Ctrl+1`〜`Ctrl+9`** | **タブ番号で直接切り替え** |
| `Cmd+H/L` | 画面端でタブ切り替え (左/右) |
| `Cmd+Shift+[` | タブを左に並べ替え |
| `Cmd+Shift+]` | タブを右に並べ替え |

### レイアウト

| キー | 操作 |
|---|---|
| `Cmd+[` | 前のスワップレイアウト |
| `Cmd+]` | 次のスワップレイアウト |

## Pane モード (`Cmd+P`)

| キー | 操作 | 自動で Normal に戻る |
|---|---|---|
| `h/j/k/l` | フォーカス移動 (左/下/上/右) | いいえ (連続操作可) |
| `p` | 次のペインへフォーカス順送り | いいえ |
| `n` | 新規ペイン (自動方向) | はい |
| `d` | 新規ペイン (下) | はい |
| `r` | 新規ペイン (右) | はい |
| `s` | 新規スタックドペイン (同じ場所にタブ的に重ねる) | はい |
| `x` | ペインを閉じる | はい |
| `f` | フルスクリーン切替 | はい |
| `z` | ペイン枠線の表示切替 | はい |
| `w` | フローティング表示/非表示 | はい |
| `e` | 埋め込み ↔ フローティング切替 | はい |
| `i` | ピン留め切替 (フローティング専用: 常に最前面に固定。ログ監視等に便利) | はい |
| `c` | ペイン名変更 (サブモードへ遷移) | — |

---

## Tab モード (`Cmd+T`)

| キー | 操作 |
|---|---|
| `n` | 新規タブ |
| `x` | タブを閉じる |
| `c` | タブ名変更 (Pane モードと統一) |
| `h/l` | 前/次のタブ |
| `1`〜`9` | タブ番号で切り替え |
| `Tab` | 直前のタブに戻る |
| `s` | 入力同期 (全ペインに同じ入力) |
| `b` | ペインを新しいタブに分離 |
| `[` | ペインを左の新タブに分離 |
| `]` | ペインを右の新タブに分離 |

---

## Scroll モード (`Cmd+Shift+S`)

| キー | 操作 |
|---|---|
| `j/k` | 1行 下/上 |
| `d/u` | 半ページ 下/上 |
| `Ctrl+F` / `PageDown` / `l` / `→` | 1ページ下 |
| `Ctrl+B` / `PageUp` / `h` / `←` | 1ページ上 |
| `s` | 検索開始 → 文字入力 → `Enter` で Search モードへ |
| `e` | エディタでスクロールバック編集 |
| `Ctrl+C` | 最下部に戻って Normal へ |

### Search モード

操作手順: `Cmd+S` → `s` → 検索文字列を入力 → `Enter` で Search モードに遷移。
オプション (`c`/`w`/`o`) は Search モードに入ってから切替える (トグルするたびに結果がリアルタイム更新)。

| キー | 操作 |
|---|---|
| `n` | 次の検索結果 (下方向) |
| `p` | 前の検索結果 (上方向) |
| `c` | 大文字小文字区別トグル |
| `w` | 折返し検索トグル |
| `o` | 全単語一致トグル (ASCII のみ) |
| `j/k/d/u` 等 | Scroll モードと同じスクロール操作 |

## Session モード (`Cmd+O`)

| キー | 操作 |
|---|---|
| `w` | セッションマネージャー (詳細は下記) |
| `d` | デタッチ (セッションはバックグラウンド継続) |
| `c` | 設定画面 (リーダーキー変更 / モード動作プリセット切替。詳細は下記) |
| `p` | プラグインマネージャー |
| `l` | レイアウトマネージャー |
| `s` | Share プラグイン |
| `a` | About 画面 |

### セッションマネージャー (`w`) の詳細

`Tab` でタブ切替。3つのタブがある:

**New Session** — 新規セッション作成

| 項目 | 操作 |
|---|---|
| セッション名 | 入力して `Enter` (空欄でランダム名) |
| フォルダ指定 | `Ctrl+f` でファイルピッカーを開く |

**Attach to Session** — 既存セッションに接続

セッション一覧が表示される (名前、タブ数、ペイン数、接続ユーザー数)。検索でフィルタ可能。

| キー | 操作 |
|---|---|
| `↑/↓` | セッション選択 |
| `Enter` | 選択したセッションに接続 |
| `Ctrl+r` | セッション名リネーム |
| `Ctrl+x` | 他の接続クライアントを切断 |
| `Del` | セッションを kill |
| `Ctrl+d` | 全セッションを kill |

**Resurrect Session** — 終了済みセッションの復元

| キー | 操作 |
|---|---|
| `↑/↓` | セッション選択 |
| `Enter` | セッションを復元 |
| `Del` | 終了済みセッションを完全削除 |
| `Ctrl+d` | 全て削除 |

### 設定画面 (`c`) の詳細

`Tab` でタブ切替。2つのタブがある:

**Rebind leader keys** — モード切替 (Primary) と直接操作 (Secondary) の修飾キーを変更

| 項目 | 現在の設定 | 選択肢 |
|---|---|---|
| Primary (モード切替: Pane, Tab 等) | Super | Ctrl / Alt / Super / Shift |
| Secondary (直接操作: New Pane 等) | Alt | Ctrl / Alt / Super / Shift |

**Change mode behavior** — キーバインドプリセットの切替

| プリセット | 動作 |
|---|---|
| Default | モードに直接切替 (例: `Ctrl+p` → Pane) |
| Unlock First (non-colliding) | 先に `Ctrl+g` で解除してから切替 (例: `Ctrl+g` → `p` → Pane) |

操作: `←↓↑→` 移動 / `Space` 選択 / `Enter` 適用 / `Ctrl+a` 適用&保存 / `Ctrl+c` リセット / `Esc` 閉じる

> ※ config.kdl でカスタムキーバインドを設定済みの場合、ここでの変更と競合する可能性あり

## Resize モード (`Cmd+R`)

| キー | 操作 |
|---|---|
| `h/j/k/l` | 方向指定で拡大 |
| `H/J/K/L` | 方向指定で縮小 (Shift) |
| `=`/`+` | 均等拡大 |
| `-` | 均等縮小 |

## Move モード (`Cmd+Shift+H`)

| キー | 操作 |
|---|---|
| `h/j/k/l` | ペインをその方向に移動 |
| `n` / `Tab` | 次の位置へ移動 |
| `p` | 前の位置へ移動 |

## Tmux モード (`Cmd+B`)

> tmux 互換のキーバインド。tmux 経験者向け。

| キー | 操作 |
|---|---|
| `"` | 水平分割 (新規ペイン下) |
| `%` | 垂直分割 (新規ペイン右) |
| `h/j/k/l` / `←↓↑→` | フォーカス移動 |
| `o` | 次のペインへフォーカス順送り |
| `z` | フルスクリーン切替 |
| `x` | ペインを閉じる |
| `c` | 新規タブ |
| `,` | タブ名変更 |
| `n` | 次のタブ |
| `p` | 前のタブ |
| `Space` | 次のスワップレイアウト |
| `[` | Scroll モードへ (tmux の copy-mode 相当) |
| `d` | デタッチ |
| `Ctrl+B` | 配下のアプリに Ctrl+B を送信 |

## CLI コマンド

```fish
zellij list-sessions           # セッション一覧
zellij attach <name>           # 再接続
zellij attach --create <name>  # あれば接続、なければ作成
zellij delete-all-sessions -y  # 全セッション削除
zellij action query-tab-names  # タブ名一覧
```

## 注意点

- **Ctrl+Q は無効** — 終了は `Cmd+O` → `d` (デタッチ) か `Cmd+Q` (Ghostty 終了)
- **vim/neovim との衝突** — モード切替を Cmd キーに変更済みのため、Ctrl 系との衝突は解消。必要に応じて `Cmd+G` で Locked モードに入る
- **compact レイアウト** — モードのキーヒントが表示されないので、このチートシートを参照
- **Ctrl+3 と日本語 IME** — macOS 日本語入力が Ctrl+3 を「顔文字を表示」として横取りする。Ghostty config で `text:` バイパス済みでタブ切替は動作するが、顔文字パネルも出る。未解決: App Shortcuts で英語メニュー名 "Show Facemarks" での退避を要検証（要再起動）
