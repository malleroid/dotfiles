# hunk cheatsheet

## ファイル・hunk 間移動

| Key       | Action             |
|-----------|--------------------|
| `,` / `.` | 前/次のファイル     |
| `[` / `]` | 前/次の hunk        |
| `{` / `}` | 前/次のコメント     |

## スクロール

| Key                    | Action                        |
|------------------------|-------------------------------|
| `j` / `k`             | 1行ずつ上下                    |
| `Space` / `f`         | ページダウン                   |
| `b` / `Shift+Space`   | ページアップ                   |
| `d` / `u`             | 半ページ上下                   |
| `g` / `G`             | 先頭 / 末尾                    |
| `←` / `→`             | コード横スクロール              |
| `Shift+←` / `Shift+→` | コード横スクロール（8列ずつ）    |

## 表示切り替え

| Key | Action                    |
|-----|---------------------------|
| `1` | split レイアウト           |
| `2` | stack レイアウト           |
| `0` | auto レイアウト            |
| `s` | サイドバー                 |
| `t` | テーマセレクタ              |
| `a` | AI アノテーション           |
| `l` | 行番号                     |
| `w` | 行折り返し                  |
| `m` | hunk メタデータ             |
| `M` | メニューバー                |
| `z` | 変更なしコンテキスト         |

## 操作

| Key    | Action                  |
|--------|-------------------------|
| `F10`  | メニューを開く           |
| `Tab`  | ファイル一覧/フィルタ切替 |
| `/`    | ファイルフィルタ          |
| `e`    | $EDITOR でファイルを開く  |
| `c`    | レビューノート作成        |
| `r`    | リロード                 |
| `q`    | 終了                    |
| `?`    | ヘルプ                   |

## マウス

| Action         | Operation    |
|----------------|--------------|
| ホイール        | 縦スクロール  |
| Shift+ホイール  | 横スクロール  |
